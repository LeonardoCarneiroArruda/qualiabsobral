O seguinte sistema foi feito para análise dos dados da pesquisa do Questionário QualiAB - Sobral, todo a lógica foi feita baseada do formato em que o LimeSurvey exporta suas respostas. O mesmo mostra todos os relatórios necessários para uma interpretação da qualidade das unidades básicas de saúde da cidade de Sobral.
Responsáveis pelo projeto: Raissa Mont'Alverne e Leonardo Carneiro
IMPORTANTE: Para que as respostas e os pontos de cada Candidato seja calculada, é necessário que visualize pelo menos 1 vez a resposta detalhada de cada pergunta.
